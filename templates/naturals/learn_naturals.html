Learning the Basics: Naturals

Basic Information

Naturals are played on the white keys of the piano!
The notes are: C, D, E, F, G, A, B, C
Between each natural note, it is a whole step, except:
E → F and B → C are half steps
Remember: If there is a black key between notes = whole step, if not = half step
